# Coassistent

## Setting Up the Development Environment with Codespaces

This project uses a devcontainer to set up the development environment with PowerShell, .NET, and React. Follow the instructions below to get started.

### Prerequisites

- [Visual Studio Code](https://code.visualstudio.com/)
- [Docker](https://www.docker.com/)
- [GitHub Codespaces](https://github.com/features/codespaces)

### Using the Devcontainer

1. Clone the repository:
   ```sh
   git clone https://github.com/ALTrijssenaar/Coassistent.git
   cd Coassistent
   ```

2. Open the repository in Visual Studio Code:
   ```sh
   code .
   ```

3. When prompted, reopen the repository in the devcontainer.

4. The devcontainer will automatically set up the development environment with PowerShell, .NET, and React.

### Dependencies

#### PowerShell

- PowerShell Core or Windows PowerShell
- PowerShell modules as required by the project

#### .NET

- .NET SDK
- .NET runtime
- Any additional .NET libraries or packages required by the project

#### React

- Node.js
- npm (Node Package Manager) or yarn
- React library
- Any additional React libraries or packages required by the project
